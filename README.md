# flask_boiler
Boiler code for flask app

# Requirements

flask

## Installation

### Install git
`sudo apt-get install python3-pip git`

### Install flask

`pip3 install flask`

# Running the script

1. clone the repo `git clone https://github.com/adamshamsudeen/flask_boiler.git`
2. `cd flask_boiler`
3. `python3 app.py`
